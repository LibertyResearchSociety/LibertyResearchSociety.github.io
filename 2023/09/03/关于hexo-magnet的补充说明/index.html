问题一：标签不显示

如图片所示，只有“查看更多”，没有标签

解决办法

根目录config.yml配置项

display

问题出在display选项里面的 - name: Name ，Name应为你博客tags或categories中存在的tags名字或categories名字，如果 - name: Name的Name是你博客tags或categories中不存在的，那么就不会显示tags或categories，就出现了图片中的问题

问题二：点击查看更多跳转至其他网页

如图片所示，点击后跳转到Example网页

解决办法

根目录config.yml配置项

url

url后面的网址改为你博客的网址即可：
